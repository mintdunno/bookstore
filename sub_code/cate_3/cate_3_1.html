SQL QuickStart Guide

Walter Shields

10. 79$

Description


SQL QuickStart Guide is a book that teaches the basics of SQL, a programming language that is used to communicate with databases. The book is written by Walter Shields, a SQL expert and mentor, and published by ClydeBank Media LLC in 2019. The book covers topics such as the structure of databases, how to use SQL to retrieve and understand data, the most important SQL queries, and how to apply SQL skills in professional settings. The book is designed for beginners who want to learn SQL in a simple and practical way. The book also includes free access to a sample database, a SQL browser app, comprehension quizzes, and other digital resources
